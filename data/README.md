# Data Directory

Questa cartella contiene dati di esempio e dataset per il progetto Nostradamus.

## Struttura

- `sample/` - Dati di esempio per tutorial e test
- `raw/` - Dati grezzi (non processati)
- `processed/` - Dati processati pronti per l'uso

## Nota

I file di dati non sono tracciati da Git per default (vedi `.gitignore`).
Se hai bisogno di condividere dataset specifici, assicurati di aggiungerli
esplicitamente al repository.
